简介

我是中国美术学院的一名大三在读学生，正在学习各种编程语言及开源框架，有一些 OpenFrameworks 和 Processing 的编程经验，有一些网络前端开发知识，现在正在学习更多的编程语言和计算机知识，比如 Javascript 以及 NodeJs 以及一些算法知识。

大概做过些什么

个人尝试过绘画，书法等传统美学训练，尝试过戏剧表演，尝试过 Audio/Visual，写过几个网站样本， 写过一些创意编程的视觉程序，尝试了一些算法声音生成等。我热爱动态声音视觉艺术， 声音艺术，以及网络艺术。

现在正沉迷于编程，以及佛经。
